# 擢升一瞥 - 游戏介绍

## 游戏概述

《擢升一瞥》是一个由AI驱动的聊天冒险游戏。在这个游戏中，玩家将通过与AI的对话来探索一个神秘的世界，每一个选择都将影响故事的发展。游戏的核心在于玩家与AI的互动，以及玩家如何通过自己的选择来塑造游戏世界。

## 基本功能

1. **AI驱动的对话系统**：玩家可以与AI进行自然语言对话，AI将根据玩家的选择和世界设定来生成响应。

2. **动态故事发展**：玩家的每一个选择都将影响故事的走向，产生不同的结局。

3. **可扩展的世界设定**：游戏的世界设定、角色信息等都存储在JSON文件中，方便开发者进行扩展和修改。

4. **游戏配置**：玩家可以在设置页面调整AI模型、游戏速度等参数。

5. **数据管理**：支持导出和导入游戏数据，方便玩家保存和分享游戏进度。

## 文件结构

- **index.html**：游戏的欢迎页面，玩家从这里开始游戏。
- **chat.html**：游戏的主界面，玩家与AI进行对话的地方。
- **prompt.json**：存储游戏的世界设定、角色信息等。
- **settings.html**：游戏的设置页面，玩家可以在这里调整游戏参数。
- **Readme.txt**：游戏的说明文档。

## 使用说明

1. **开始游戏**：打开index.html文件，点击"开始游戏"按钮进入游戏界面。

2. **与AI对话**：在chat.html页面的输入框中输入消息，点击发送按钮或按回车键发送消息，AI将生成响应。

3. **调整设置**：点击页面右上角的"设置"按钮进入设置页面，调整游戏参数。

4. **返回首页**：点击页面左上角的"返回首页"按钮返回欢迎页面。

## 开发指南

### 扩展世界设定

游戏的世界设定存储在prompt.json文件中，开发者可以通过修改这个文件来扩展游戏内容：

- **gameInfo**：游戏的基本信息，如标题、版本等。
- **worldSetting**：游戏的世界背景、规则和地点信息。
- **characters**：游戏中的角色设定。
- **initialDialogue**：游戏的初始对话。
- **dialogueRules**：AI对话的规则。
- **gameEngine**：游戏引擎的配置和命令。
- **extensions**：游戏的扩展功能，如成就系统、支线任务等。

### 接入AI API

目前游戏使用的是模拟AI回复，开发者可以通过修改chat.html中的getAIResponse函数来接入真实的AI API：

```javascript
function getAIResponse(userMessage) {
    // 这里应该是调用AI API的地方
    // 目前使用简单的模拟回复
    const responses = [
        "这是一个有趣的选择。让我想想...",
        "你的话让我陷入了沉思。",
        "我理解你的想法。接下来你想做什么？",
        "这个世界充满了可能性，你可以继续探索。",
        "你的选择将影响故事的发展。"
    ];
    return responses[Math.floor(Math.random() * responses.length)];
}
```

### 扩展游戏功能

开发者可以通过修改chat.html和settings.html来扩展游戏功能，如添加物品系统、成就系统等。

## 技术栈

- **前端**：HTML + CSS + JavaScript（原生，不使用框架）
- **AI集成**：预留接口，方便后续接入
- **数据存储**：JSON文件（用于配置和世界设定）
